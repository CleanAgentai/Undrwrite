# Undrwrite — How It Works

## Overview

Vienna is Franco's AI executive assistant at Private Mortgage Link. She handles broker communications, collects documents, analyzes deals, and sends Franco everything he needs to make decisions — all via email.

Franco stays in control through a draft approval system: Vienna never sends an email to a broker without Franco seeing it first (for escalated and reviewed deals).

---

## Deal Statuses

| Status | What It Means |
|--------|---------------|
| **Active** | Deal is open — Vienna is collecting docs and info from the broker |
| **LTV Escalated** | LTV is over 80% — waiting on Franco to approve or reject |
| **Under Review** | LTV is 80% or under — Franco is reviewing the preliminary file |
| **Completed** | Deal is done |
| **Rejected** | Deal was declined |

---

## Step-by-Step Workflow

### Step 1: Broker Sends First Email

A broker emails in with a loan request (and sometimes attachments like applications, credit reports, appraisals, etc.).

**What Vienna does:**
- Reads the email and all attachments
- Creates a deal file
- Sends a welcome email introducing herself as Franco's executive assistant
- Only asks for what's MISSING — she won't ask for things the broker already provided

**Forms attached to the first email:**
- PNW Statement Form — always included
- Loan Application Form — only if the broker didn't already send their own application
- Borrower Intake Form is NOT sent yet (sent later after Franco approves)

**What Vienna asks for (only if not already provided):**
- Exit strategy
- What is owing on existing mortgages
- Current appraisal
- Proof of income / NOA
- Credit bureau reports
- Loan amount

---

### Step 2: Broker Replies

When the broker replies, Vienna reads the full conversation and all documents on file, then responds naturally.

- If the broker asks a question, Vienna answers it
- If the broker sends new documents, Vienna acknowledges them
- Vienna only asks for what's still missing

**Once LTV is confirmed:**

- **LTV over 80%** → Deal escalated to Franco with a summary and all documents as a zip file. Vienna does NOT reply to the broker — Franco decides first.

- **LTV 80% or under** → Franco gets a preliminary review with a full deal summary and all documents as a zip file. Vienna does NOT reply to the broker — Franco decides first.

- **LTV not yet known** → Vienna keeps the conversation going with the broker.

---

### Step 3: Franco Reviews the Deal

Franco receives an email with:
- Full deal summary (borrower info, loan details, risks, exit strategy, deal rating)
- List of documents received and what's still missing
- All documents attached as a zip file
- The full email conversation with the broker

**Franco replies with one of:**

1. **"Approved"** — Vienna drafts a doc request email for the broker
2. **"Rejected"** or **"Pass"** — Vienna drafts a rejection email
3. **Anything else** (conditions, questions, instructions) — Vienna drafts a professional email based on Franco's notes

---

### Step 4: Draft Approval (Human-in-the-Loop)

After Franco replies, Vienna generates the email and sends Franco a **preview** in the same email thread.

Franco then replies:
- **"Send"** or **"Looks good"** → Email goes to the broker as-is
- **Any edits** (e.g., "make it shorter" or "remove the part about income") → Vienna revises the email and sends it

This way, Franco always sees exactly what the broker will receive before it goes out.

**After the email is sent:**
- If approved → deal goes back to **Active** (Vienna continues collecting remaining docs)
- If rejected → deal goes to **Rejected**
- If conditions → deal stays in review, broker receives Franco's conditions

---

### Step 5: Document Collection

After Franco approves, Vienna continues working with the broker to collect all remaining documents. The Borrower Intake Form is attached at this stage.

Vienna tracks what's been received and what's still needed, and follows up conversationally.

---

### Step 6: Follow-Up Reminders (Automatic)

If a broker goes silent for more than 2 days, Vienna sends a friendly follow-up reminder.

- **Reminder 1:** Casual check-in
- **Reminder 2:** A bit more direct
- **Reminder 3:** "We'll close the file for now, but feel free to reach out anytime"

Maximum 3 reminders per deal. After that, it's Franco's call.

---

### Step 7: Daily Summary (9 PM MST)

Franco receives a daily email every evening with:
- Deals that need his attention (escalated or under review)
- New emails received in the past 24 hours
- All active deals and their current status
- Deals that have gone quiet (3+ days no activity)
- Which deals got follow-up reminders that day
- Deals that have hit the maximum 3 reminders

---

## Broker Replies During Franco's Review

If a broker sends new information while Franco is reviewing a deal, Vienna does NOT reply. Instead:
- The broker's message is forwarded to Franco as a "[Broker Update]"
- Franco has the latest info when making his decision

## Required Documents

### Personal Deals

| Document | Notes |
|----------|-------|
| Government-Issued ID | |
| Property Appraisal | |
| Property Tax Assessment | Current balance |
| Notice of Assessment (NOA) | For all borrowers |
| Current Mortgage Balance Statement | |
| Income Verification | Proof of income |
| Credit Bureau Report | If not included, Vienna asks "Have you pulled credit?" |
| Loan Application Form | Broker's own application is accepted |
| PNW Statement (Personal Net Worth) | Broker's own version is accepted |

### Corporate Deals (same as above, plus)

| Document | Notes |
|----------|-------|
| Corporate Financial Statements | 2023, 2024, 2025 |
| T1s for Key Principals | 2023, 2024 |
| Borrower Resume / Development Experience | If applicable |

### Other Documents (accepted but not required)

These are acknowledged if received but Vienna won't ask for them:

- AML Report
- PEP Report
- Insurance
- Title Search
- Survey
- Environmental Report

## Deal Rating System

When Franco receives the deal summary, each deal is rated:

| Rating | What It Means |
|--------|---------------|
| **GREAT** | Conservative LTV (60% or less), strong property, clear exit strategy, clean file |
| **OKAY** | One or two weak areas but not fatal — higher LTV, credit issues, but exit strategy is believable |
| **WEAK BUT WORKABLE** | Multiple risks (high LTV, poor credit, low income) but at least one strong factor (great property, additional collateral) |

---

## How Vienna Communicates

- Introduces herself as Franco's executive assistant
- Addresses brokers by first name
- Warm, friendly, and concise — not robotic or corporate
- Never repeats back what the broker already said
- Answers questions before asking for documents
- If credit reports are missing, asks "Have you pulled credit?"
- If forms come back blank, tells the broker exactly which ones are blank
- LTV is confirmed from the appraisal, not the application form
- MLS listing and appraisal are two different things — never confused
